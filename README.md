## Introduction

icampus-manager는 아이캠퍼스의 기능을 확장해주는 크롬 익스텐션입니다.
현재 아래와 같은 기능을 제공하고 있습니다.
- 강의 및 과제 목록 보기
- 강의 자동 재생
- 강의 자료 일괄 다운로드
- 사용자 설정 및 강의 추가

## Prerequisites
최신 버전의 Chrome 브라우저가 설치되어 있어야 합니다.
